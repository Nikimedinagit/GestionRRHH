Listado y Control de Tipos de Criterios

Panel para administrar y consultar tipos de criterios.

Información de Ayuda

Criterio Activo

El criterio está disponible para ser asignado en evaluaciones. Se muestra con ícono verde y opciones visibles.

Criterio Desactivado

El criterio está inactivo y no puede ser asignado. Se muestra con ícono rojo y opciones ocultas.

Editar

Permite modificar el nombre del tipo de criterio seleccionado.

Activar / Desactivar

Permite habilitar () o deshabilitar () el criterio para su uso en evaluaciones.
